# 无创产前胎儿性别鉴定 qPCR 技术方案

> **适用范围**：X连锁遗传病携带者家庭、CAH风险家庭等医学指征明确的非侵入性产前诊断  
> **目标性能**：孕6周检出率 ≥95%，孕7周准确率 ≥98%  
> **最后更新**：2026-04-30

---

## 一、技术原理

母体血浆中存在游离胎儿DNA（cell-free fetal DNA, cffDNA），占母体总血浆游离DNA的 **3%-20%**，孕周越早比例越低。胎儿性别鉴定通过检测Y染色体特异性序列实现：

- **男胎**：可检出Y染色体靶标（TSPY/SRY/DYS14等）
- **女胎**：无Y染色体靶标信号，仅检出母体常染色体参照

### 1.1 靶标选择策略

| 靶标 | 染色体位置 | 拷贝数 | 灵敏度 | 特异性 | 适用场景 |
|------|-----------|--------|--------|--------|----------|
| **TSPY** | chrY ~9.55 Mbp | 35-60拷贝 | ★★★★★ | ★★★★☆ | **6-7周早期检测首选** |
| **YRS** | chrY 多区域重复 | 高拷贝 | ★★★★★ | ★★★★☆ | 早期检测备选 |
| **SRY** | chrY:27.86 Mbp | 1拷贝 | ★★★☆☆ | ★★★★★ | **特异性确认靶标** |
| **DYS14** | chrY 重复区 | 多拷贝 | ★★★★☆ | ★★★★☆ | 可配合SRY使用 |

**推荐方案**：**TSPY（初筛/高敏）+ SRY（确认/特异）** 双靶标组合。

## 二、引物与探针设计

### 2.1 设计原则（TaqMan MGB）

| 参数 | 引物 Forward/Reverse | 探针 Probe |
|------|---------------------|-----------|
| **长度** | 18-22 nt（优选20） | 18-21 nt |
| **Tm (Wallace法则)** | 57-64°C | 64-72°C |
| **GC含量** | 35-65% | 35-70% |
| **扩增子大小** | 70-130 bp（cffDNA优选<100） | - |
| **标记方式** | - | 5'-FAM/VIC-MGB-NFQ-3' |

**设计禁忌**：
- 避免5'端以G开头（探针荧光猝灭）
- 避免连续4个相同碱基（AAAA、GGGG等）
- 避免引物3'端与其他引物/探针互补（防止引物二聚体）
- 探针Tm应比引物Tm高5-10°C

---

### 2.2 TSPY 靶标（初筛管，FAM通道）

> **靶基因**：TSPY1/TSPY2 保守区  
> **参考基因组**：GRCh38 chrY ~9,551,000-9,551,500

| 元件 | 序列（5'→3'） | 长度 | Tm |
|------|--------------|------|----|
| Forward Primer | `CCTCTAAAGTTGTAGAGTGAAT` | 22 bp | ~60°C |
| Reverse Primer | `TTTCGTGTTGAATTCACATGTT` | 22 bp | ~58°C |
| TaqMan Probe (FAM) | `TGGAATGTGTTGAGATTCTTC` | 21 bp | ~66°C |
| **预期扩增子** | - | **~85 bp** | - |

> ⚠️ **验证要求**：必须用 NCBI Primer-BLAST 验证，确保仅匹配 chrY TSPY1/TSPY2/TSBP，排除 chrX TSPY-like 假基因（TSPYL1/TSPYL2）交叉扩增。

---

### 2.3 SRY 靶标（确认管，FAM通道）

> **靶基因**：SRY HMG-box 外显子1  
> **参考基因组**：GRCh38 chrY:27,868,555-27,876,328

#### 经典引物（文献常用，扩增子偏长）

| 元件 | 序列（5'→3'） | 长度 | 说明 |
|------|--------------|------|------|
| Forward | `CATGAACGTGGATCAGCAG` | 19 bp | Rijnders 2003 |
| Reverse | `CTCCCAGGCATGTCAGAG` | 18 bp | 需确认方向 |
| 扩增子 | - | ~152 bp | 对cffDNA偏长，建议优化 |

#### 现代短扩增优化方案（推荐）

基于 chrY:27,869,800-27,872,000 区段设计：

| 元件 | 序列（5'→3'） | 长度 | Tm |
|------|--------------|------|----|
| Forward Primer | `TAGCCAATGTTACCCGATTGT` | 21 bp | ~60°C |
| Reverse Primer | `TTAAAGAGGAAACATGTGAATT` | 22 bp | ~58°C |
| TaqMan Probe (FAM/VIC) | `CGGTCCCGTTGCTGCGGTGA` | 20 bp | ~68°C |
| **预期扩增子** | - | **~90 bp** | - |

> ⚠️ 上述序列为**设计候选序列**，正式使用前必须通过以下验证：
> 1. NCBI Primer-BLAST 全基因组特异性扫描
> 2. **dbSNP check**：确认靶区无已知人群SNP（避免假阴性）
> 3. 梯度稀释灵敏度实验（至少3个男性cfDNA样本，10倍梯度稀释至0.1 GE/mL）

---

### 2.4 内参靶标（总DNA质控，VIC通道）

> **靶基因**：GAPDH 或 TERT  
> **用途**：验证血浆cfDNA提取质量、PCR反应效率、排除抑制剂干扰

推荐 GAPDH 短扩增子方案：

| 元件 | 序列（5'→3'） | 长度 | Tm |
|------|--------------|------|----|
| Forward Primer | `AATCCCATCACCATCTTCCA` | 20 bp | ~58°C |
| Reverse Primer | `TGGACTCCACGACGTACTCA` | 20 bp | ~60°C |
| TaqMan Probe (VIC) | `CCTCCCCGCTTCGCTGTCTGC` | 21 bp | ~70°C |
| **预期扩增子** | - | **98 bp** | - |

## 三、实验方案

### 3.1 样本要求

| 项目 | 要求 |
|------|------|
| **样本类型** | 母体外周血，EDTA抗凝管或Streck管采集 |
| **采血量** | ≥10 mL（推荐2×10 mL） |
| **血浆分离体积** | ≥1 mL（每例推荐2 mL以上） |
| **运输条件** | 室温24h内送达，或4°C 48h内；避免反复冻融 |
| **溶血检测** | **禁止溶血样本**（Hb > 500 mg/dL会导致母体DNA背景升高） |
| **孕周确认** | 以超声测量顶臀径（CRL）或末次月经日期确认 |

> **关键操作**：采血后需在6h内完成血浆分离（两次离心：1,600g 10min 去细胞 → 16,000g 10min 去残细胞）。

---

### 3.2 cfDNA提取

推荐使用经NIPT验证的自动化提取方案：

| 推荐平台 | 货号/试剂 | 特点 |
|---------|----------|------|
| QIAGEN | QIAamp Circulating Nucleic Acid Kit | 金标准，1-5 mL血浆 |
| Roche | MagNA Pure 96 System + cfNA Kit | 高通量自动化 |
| 国产替代 | 华大智造/达安等cfDNA提取试剂 | 成本优化 |

**提取后质控**：
- Nanodrop/荧光法定量（Qubit dsDNA HS Kit）
- cfDNA片段分析：主峰应在160-170 bp（对应核小体片段）
- 若>500 bp大分子DNA比例>20%，提示溶血/细胞破裂污染

---

### 3.3 qPCR 反应体系

#### 初筛管（TSPY + GAPDH）

| 组分 | 体积 | 终浓度 |
|------|------|--------|
| TaqMan Fast Advanced Master Mix | 10.0 μL | 1× |
| TSPY Forward Primer (10 μM) | 0.9 μL | 450 nM |
| TSPY Reverse Primer (10 μM) | 0.9 μL | 450 nM |
| TSPY Probe (5 μM, FAM) | 0.5 μL | 125 nM |
| GAPDH Forward Primer (10 μM) | 0.9 μL | 450 nM |
| GAPDH Reverse Primer (10 μM) | 0.9 μL | 450 nM |
| GAPDH Probe (5 μM, VIC) | 0.5 μL | 125 nM |
| 模板 cfDNA | 5.0 μL | ~1/4血浆当量 |
| Nuclease-free Water | 补足至 20 μL | - |

#### 确认管（SRY + GAPDH）

组装方式同上，仅将TSPY引物探针替换为SRY引物探针（FAM通道），GAPDH作为共用内参（VIC通道）。

> **注**：两管可合并为多重反应（TSPY-FAM + SRY-NED + GAPDH-VIC），但需通过引物/探针浓度优化避免竞争抑制。对于早期孕周低丰度cffDNA，建议分单管检测以最大化灵敏度。

---

### 3.4 循环参数

适用平台：ABI StepOne/QuantStudio 系列、Roche LightCycler 等

| 步骤 | 温度 | 时间 | 循环数 |
|------|------|------|--------|
| 预变性 | 95°C | 20 s | 1 |
| 变性 | 95°C | 3 s | |
| 退火/延伸 | **60°C** | **30 s** | 45 |

> **关键设置**：采用 **45个循环**（非常规40循环），以捕获极低拷贝的cffDNA信号。数据采集在60°C退火阶段进行。

---

### 3.5 对照设置（每板必设）

| 对照类型 | 数量 | 目的 |
|----------|------|------|
| NTC (No Template Control) | ≥2孔 | 排除气溶胶/试剂污染 |
| 男性阳性对照 | ≥2孔 | 已知男性cfDNA或质粒标准品 |
| 女性阴性对照 | ≥2孔 | 已知女性cfDNA或水 |
| 低浓度阳性对照 | ≥1孔 | 接近检出限的稀释标准品（如10 GE/mL） |

## 四、结果判读

### 4.1 阈值设定

| 靶标 | 阈值基线 | 手动阈值 | 说明 |
|------|---------|---------|------|
| TSPY | 3-15 cycle | ΔRn = 0.02 | 多拷贝信号强，阈值可略高 |
| SRY | 3-15 cycle | ΔRn = 0.02 | 单拷贝信号弱，需关注低荧光增幅 |
| GAPDH | 自动/手动 | ΔRn = 0.1 | 内参信号应强且一致 |

### 4.2 Ct值判读规则

| 靶标 | 阳性判读 | 阴性判读 | 灰区/需复查 |
|------|---------|---------|------------|
| TSPY | Ct ≤ 38 | Ct > 38 或无扩增 | Ct 36-38且曲线不佳 |
| SRY | Ct ≤ 38 | Ct > 38 或无扩增 | Ct 36-38且曲线不佳 |
| GAPDH | Ct ≤ 35 | Ct > 35 | 内参异常时整板不合格 |

### 4.3 综合判定矩阵

| 初筛管 (TSPY) | 确认管 (SRY) | 内参 (GAPDH) | **最终判定** | 处理建议 |
|--------------|-------------|-------------|-------------|---------|
| 阳性 (Ct<38) | 任意 | 正常 (Ct<35) | ♂ **男胎** | 可直接报告 |
| 阴性 | 阳性 (Ct<38) | 正常 | ♂ **男胎** | SRY特异性确认 |
| 阴性 | 阴性 | 正常 | ♀ **女胎** | 无Y信号 |
| 阴性 | 阴性 | 异常 (Ct>35) | ❌ **不合格** | 提取失败/PCR抑制，需重新采血 |
| 阳性 | 阴性 | 正常 | ⚠️ **边界/低丰度** | ≤6周时加注" Y信号低，建议7周后复查" |
| 任意 | 任意 | 任意 | ❌ **NTC阳性** | 污染，整批重做 |

### 4.4 孕周特异性置信度标注

| 孕周 | TSPY检出预期 | 结果标注 |
|------|-------------|---------|
| ≤5周 | ~50-70% | **不推荐检测**，假阴性风险高 |
| 6周 | ~95% | 可报告，低信号样本加注"建议复查" |
| 7周 | ~98% | 可可靠报告 |
| ≥8周 | ~99% | 高置信度，与NIPT常规方案一致 |

## 五、性能验证

### 5.1 方法学验证要求

| 验证项目 | 最低要求 | 合格标准 |
|---------|---------|---------|
| **分析灵敏度** | LOD实验：男性DNA梯度稀释（50→0.1 GE/mL） | LOD ≤ 1.0 GE/mL，且95%检出 |
| **分析特异度** | 女性cfDNA ≥30例 | 假阳性率 = 0% |
| **临床灵敏度** | 已知性别孕早期样本（6-7周）≥60例 | ≥95%（6周）、≥98%（7周） |
| **临床特异度** | 已知女性孕早期样本 ≥60例 | ≥99% |
| **精密度** | 低/中/高浓度重复检测（n=20×3天） | CV(Ct) < 5% |
| **干扰实验** | 溶血、脂血、黄疸样本 | 内参Ct变化 < 2 |
| **携带污染** | 高浓度阳性→阴性连续检测 | 阴性样本无扩增 |

### 5.2 ddPCR 方法学比对（可选）

如实验室配备 Bio-Rad QX200 或 RainDrop 系统，建议：
- 取20例6-7周qPCR边界样本进行ddPCR平行检测
- 文献证实 ddPCR 灵敏度更高（D'Aversa 2018 在<7周达到100%）
- 可作为 qPCR 灰区结果的确证手段

## 七、伦理与合规

### 7.1 医学指征限定

本检测仅限以下医学指征：

1. **X连锁隐性遗传病**携带者妊娠：
   - 假肥大型肌营养不良（DMD）
   - 血友病A/B
   - 脆性X综合征
   - 肾上腺脑白质营养不良等

2. **X连锁显性遗传病**风险评估：
   - Rett综合征（多数为新发突变，少数为母系遗传）
   - 色素失禁症等

3. **先天性肾上腺皮质增生症（CAH）**：
   - 21-羟化酶缺乏症女性胎儿需产前地塞米松治疗

4. **其他医学需要**：如性发育异常（DSD）家族史

### 7.2 禁止非医学需要的性别鉴定

> 根据中国《禁止非医学需要的胎儿性别鉴定和选择性别人工终止妊娠的规定》，严禁非医学需要的胎儿性别鉴定。本方案仅限上述医学指征，且需获得：
> - 产前诊断资质医疗机构执业许可
> - 遗传咨询门诊医师开具的申请单
> - 孕妇签署的知情同意书

### 7.3 知情同意要点

知情同意书必须明确告知：
- 检测原理及局限性（假阴性/假阳性风险）
- 6周结果可能存在不确定性，建议复查
- 极罕见情况下（如母体移植男性干细胞、近期输注男性血液），可能出现假阳性
- XX男性（SRY易位）和XY女性（SRY缺失）等性发育异常可导致结果与表型不符
- 检测仅用于医学目的，不用于性别选择

## 八、常见问题（FAQ）

**Q1：为什么6周检测准确率达不到100%？**  
A：孕6周时cffDNA在母体血浆中浓度极低（约10-50 GE/mL），且片段化严重。TSPY多拷贝靶标虽提高了灵敏度，但仍有约5%的样本因cffDNA量低于检测限而出现假阴性。7周后胎盘滋养层大量凋亡，cffDNA浓度上升，准确率可稳定达到98%以上。

**Q2：可以使用NIPT测序平台检测性别吗？**  
A：可以，且准确率更高（>99%）。但NIPT测序成本高、周期长（3-5天）。本qPCR方案可在3小时内出结果，成本仅为测序的1/10，适合快速预筛和医学指证明确的小批量样本。

**Q3：双胎妊娠如何判读？**  
A：若检出Y信号，至少有一胎为男性；若无Y信号，两胎均为女性。无法通过本方案确定双胎是否一男一女。

**Q4：体外受精（IVF）妊娠是否适用？**  
A：适用。但需注意卵胞浆内单精子注射（ICSI）可能略微影响cffDNA动力学，建议在7周后检测。

**Q5：母体BMI是否影响结果？**  
A：母体肥胖（BMI>30）会导致血浆体积增加而cffDNA浓度相对降低，可能导致假阴性率升高。对于肥胖孕妇建议增加血浆用量或延迟至8周后检测。

## 十、重复实验与矛盾结果判读手册（生产实践版）

> **适用范围**：临床常规检测中同一批次内重复实验结果不一致、或TSPY/SRY/GAPDH多基因间结果冲突时的系统性判读  
> **核心原则**："重复是为了排除随机误差，矛盾是为了暴露系统性偏差"

---

### 10.1 重复实验的实操规范

#### 10.1.1 重复次数设置

| 检测阶段 | 重复次数 | 重复方式 | 目的 |
|---------|---------|---------|------|
| **初筛管（TSPY）** | 每样本 **3孔** | 同一反应板内3个独立孔 | 评估PCR稳定性、排除加样误差 |
| **确认管（SRY）** | 每样本 **3孔** | 同一反应板内3个独立孔 | 确认男性特异性信号 |
| **内参（GAPDH）** | 每样本 **2孔** | 初筛管/确认管各1孔 | 双重质控 |
| **重测规则** | 任一靶标3孔中≥1孔出现Ct差值>2 | 整板重测 | 排除反应条件波动 |

> **为什么是3孔而非2孔？**  
> 2孔时若出现"一阴一阳"，无法判断哪一个是真阳/真阴；3孔时以"多数原则"（2/3）解决绝大多数边界矛盾，且第3孔可为ddPCR抽查保留等分样本。

#### 10.1.2 分板逻辑（避免批次效应）

| 场景 | 分板策略 |
|------|---------|
| 标准检测 | 每板≤48例待检样本 + 4例对照（2阳+2阴+2NTC），剩余空孔用NTC或水填充 |
| 重复验证 | 同一原始提取液分装至 **A板**（初筛）和 **B板**（确认），避免交叉污染 |
| 疑难复核 | 同一原始提取液另取等分，在 **不同仪器、不同操作日** 重复检测 |

#### 10.1.3 重复孔的Ct差值判读

| Ct差值 | 判读 | 处理 |
|--------|------|------|
| ≤1.0 | **高度一致**，取中位数Ct报告 | 正常判读 |
| 1.0-2.0 | **可接受范围**，取中位数Ct | 记录备注，不触发重测 |
| 2.0-3.5 | **可疑波动**，需查看扩增曲线 | 若曲线形态明显差异（如1孔为典型S型，2孔为直线），触发重测 |
| >3.5 或一阴一阳 | **系统异常** | 必须重测整板；同时检查是否孔间加样不均、气泡、探针降解 |

> **关键细节**：3孔都应在同一行或同一列中相邻设置（如A1-A3），避免板边缘效应。绝不将3孔分散在板的不同象限。

### 10.2 重复结果一致时的判读

当3孔结果完全一致时，精简判读如下：

| 靶标 | 3孔均为阳性 | 3孔均为阴性 | 3孔均在灰区 |
|------|-----------|-----------|-----------|
| TSPY | 判"阳性"，Ct取中位数 | 判"阴性" | 标注"低信号男胎可能"，建议7周复查 |
| SRY | 判"阳性"，Ct取中位数 | 判"阴性" | 标注"低丰度"，需结合TSPY |
| GAPDH | 通过质控 | 提取/PCR失败，整板作废 | 若仅1-2孔灰区仍可接受 |

### 10.3 重复结果矛盾时的判读流程

#### 10.3.1 同一基因3孔矛盾（2阳1阴 或 1阳2阴）

**Step 1：检查是否存在"物理异常孔"**
- 查看实时扩增曲线：阴性孔是否为"直线无信号"，还是有微弱抬升但被阈值判定为阴性？
- 检查PCR管/孔是否有气泡、漏液、探针未混匀（颜色不均）
- 检查该孔是否位于板边缘（边缘孔热传导不均，Ct可能偏差0.5-1.5）

**Step 2：排除提取液不均**
- 提取后cfDNA是否充分混匀？高浓度cffDNA可能沉降，导致分装时浓度不一致
- 对策：重新震荡混匀提取液，取第2批3孔重测

**Step 3：多数原则 + 保守原则**

| 矛盾模式 | 多数结果 | 最终判定 | 置信度 | 报告标注 |
|---------|---------|---------|--------|---------|
| 2阳 + 1阴 | 阳性 | **男胎** | 高 | 记录"3孔中1孔未检出" |
| 1阳 + 2阴 | 阴性 | **女胎** | 中 | ⚠️ **加注"低丰度Y信号，建议7周后复查"** |
| 2灰区 + 1阴 | 灰区 | **边界/建议复查** | 低 | 明确建议延迟至7周后重新采血 |

> **关键原则**："宁可报告不确定，也不强行给出一个可能错误的确定性结论。"  1阳2阴时虽然技术上多数为阴性，但那个阳性孔提示cffDNA存在只是浓度临界——尤其在≤6周时，这正是假阴性的高发场景。

**Step 4：触发二次提取还是直接报告？**

- 若剩余原始血浆≥0.5 mL：**优先二次提取重测**（排除提取环节的不均）
- 若血浆已耗尽：**报告"结果不确定，建议重新采血复查"**
- 若为医学紧急指征（如CAH需立即启动地塞米松治疗）：通知临床医师，由遗传咨询医师根据风险收益决定

#### 10.3.2 不同基因之间的矛盾（TSPY vs SRY vs GAPDH）

实际生产中，最常见的矛盾组合按发生频率排序：

| 矛盾组合 | 发生概率 | 生物学/技术原因 | 判读优先级 |
|---------|---------|----------------|-----------|
| **TSPY+ / SRY-** | 最高（~5-8%@6周） | TSPY多拷贝灵敏度高于SRY单拷贝 | 见10.3.3 |
| **TSPY- / SRY+** | 低（~0.5%） | TSPY假阴性或SRY非特异性扩增 | SRY阳性优先，但需排查污染 |
| **TSPY+ / SRY+ / GAPDH-** | 罕见 | GAPDH探针失效/降解 | 整板重测 |
| **TSPY- / SRY- / GAPDH-** | 中等 | 提取失败、严重溶血、孕周过小 | 不合格，重新采血 |
| **TSPY+ / SRY+（女胎确认）** | 极低 | 母体微嵌合、移植史、近期男性血液输注 | 见10.4.1 |

#### 10.3.3 TSPY+ / SRY- 矛盾组合的详解与判读

这是孕早期检测中最典型的"预期内矛盾"，不是错误。

**为什么会发生？**

- TSPY基因在chrY上有35-60个拷贝，SRY仅1个拷贝
- cffDNA浓度极低时（如6周），PCR检测概率与靶标拷贝数成正比
- TSPY检测概率 ≈ 1-(1-p)^60，SRY检测概率 ≈ p（p为cffDNA分子被PCR捕获的单分子概率）
- 结果：TSPY在cffDNA浓度10-30 GE/mL时即可检出，SRY需>50 GE/mL

**判读流程（决策树）**：

```
TSPY+ / SRY-（3孔模式均为2-3孔阳性）
│
├─ GAPDH正常？
│   ├─ 否 → 整板不合格，重测
│   └─ 是 → 进入下一步
│
├─ 孕周确认 ≤6+0 周？
│   ├─ 是 → 判读："♂ 男胎（低丰度）"
│   │         报告加注："Y染色体信号丰度低，符合早期妊娠cffDNA特征。
│   │         如临床需要更高置信度，建议孕7-8周后复查。"
│   │         置信度：中等
│   └─ 否（≥7周）→ 进入下一步
│
├─ 二次提取重测（可选）
│   ├─ TSPY仍+ / SRY仍- → 确认男胎，但标注"SRY低信号"
│   └─ TSPY+ / SRY+ → 确认男胎，高置信度
│
└─ 临床背景
    ├─ CAH紧急治疗需求 → 可直接启动，但72h内安排复查
    └─ 非紧急（如DMD携带者知情）→ 建议复查后再做决策
```

#### 10.3.4 TSPY- / SRY+ 矛盾组合的判读

此组合出现概率低，但**风险最高**。

**可能原因分析**：

| 原因 | 概率 | 验证方法 |
|------|------|---------|
| TSPY引物区存在罕见突变/多态性导致假阴性 | 中 | 设计备选TSPY引物对重测 |
| SRY非特异性扩增（引物与chrX假基因交叉） | 中 | NTC阳性？阴性对照有信号？BLAST验证 |
| 单孔SRY污染（操作台/枪头/气溶胶） | 高 | 该板NTC和阴性对照是否阳性？ |
| 母体微嵌合（少量男性细胞） | 极低 | 母体白细胞STR测序 |
| TSPY引物降解/批次问题 | 低 | 阳性TSPY标准品是否正常？ |

**判读规则**：

- **SRY仅1孔阳性，TSPY全部阴性** → 最可能是该孔污染，整板作废重测
- **SRY 2-3孔均阳性，TSPY全部阴性** → 极罕见，需：
  1. 检查NTC和阴性对照 — 若NTC也SRY阳性 → 整板污染，彻底重做
  2. 若NTC阴性 → 用**另一个TSPY候选引物对**重测
  3. 若备选TSPY仍阴性 → 报告"异常结果，建议ddPCR验证或7周后复查"
  4. 同时启动母体cfDNA STR分析，排查微嵌合

### 10.4 矛盾结果的根因分析（RCA）

#### 10.4.1 假阳性根因树

```
检测结果为"男胎"，但最终确认为女胎（或样本本身为女）
│
├─ 技术因素
│   ├─ NTC污染（最常见）
│   │   └─ 原因：阳性扩增产物气溶胶、阳性对照开盖溅出、交叉污染
│   │   └─ 验证：同一批NTC有≥1孔阳性
│   │   └─ 对策：UNG酶防污染体系、物理隔离加样区/扩增区
│   ├─ 探针/引物非特异性扩增
│   │   └─ 原因：引物设计未排除chrX假基因
│   │   └─ 验证：女胎样本（n≥10）扩大验证，观察是否有假阳性
│   │   └─ 对策：重新设计引物，提高退火温度至62°C
│   └─ 试剂污染
│       └─ 原因：Master Mix出厂污染（极罕见但致命）
│       └─ 验证：新开封试剂盒+水+NCT运行
│
├─ 生物因素
│   ├─ 母体微嵌合（microchimerism）
│   │   └─ 原因：既往男性胎儿/输血/移植后男性干细胞残留
│   │   └─ 文献：男性DNA可在母体循环中持续数十年（Lo 1997后多项研究证实）
│   │   └─ 验证：母体白细胞STR分析显示Y信号
│   ├─ 近期男性血液输注
│   │   └─ 验证：病历查询（询问孕妇3个月内是否有输血史）
│   └─ 性发育异常（DSD）
│       └─ 46,XX SRY+ 性反转（发生率约1:20,000）
│       └─ 验证：无创或侵入性产前诊断确认核型
│
└─ 样本混淆
    └─ 原因：采血/离心/提取/加样过程中样本标签错误
    └─ 验证：母体/样本条形码核对、STR identity检验
```

#### 10.4.2 假阴性根因树

```
检测结果为"女胎"，但最终确认为男胎
│
├─ 技术因素
│   ├─ cffDNA提取失败/量不足
│   │   └─ 原因：血浆分离延迟（>6h）、反复冻融、溶血
│   │   └─ 指标：GAPDH Ct > 35 或无信号
│   │   └─ 对策：拒收溶血样本、规范血浆分离流程
│   ├─ PCR抑制
│   │   └─ 原因：提取液含EtOH残留、肝素抗凝（肝素是Taq酶强抑制剂）
│   │   └─ 指标：内参GAPDH Ct偏高，但模板DNA定量显示浓度正常
│   │   └─ 对策：EtOH充分风干、拒绝肝素管样本
│   ├─ 靶标突变导致引物/探针不匹配
│   │   └─ 原因：SRY/TSPY引物结合区存在罕见SNP/Indel
│   │   └─ 文献：SRY基因人群突变率低（<0.1%），但理论上存在
│   │   └─ 对策：多靶标并行（TSPY+SRY+DYS14），设计保守区引物
│   └─ 探针荧光淬灭/降解
│       └─ 原因：探针反复冻融、光照暴露、超过有效期
│       └─ 指标：阳性标准品Ct值漂移（比平时高3-5）
│       └─ 对策：探针分装-20°C避光保存、每6个月更换
│
├─ 生物因素
│   ├─ 孕周过小/胎盘发育延迟
│   │   └─ 原因：实际孕周<5周，cffDNA浓度低于LOD
│   │   └─ 对策：严格超声确认孕周，拒绝<5周样本
│   ├─ 母体肥胖稀释效应
│   │   └─ 原因：BMI>30，血浆体积大，cffDNA浓度相对降低
│   │   └─ 对策：增加血浆用量至2 mL以上，或延迟至8周检测
│   ├─ 胎儿染色体异常
│   │   └─ Turner综合征（45,X）：男胎但Y染色体完全缺失
│   │   └─ 验证：后续产检超声发现胎儿水肿、颈项透明层增厚
│   └─ 双胎之一消失（vanishing twin）
│       └─ 原因：初始为男双胎，一胎停止发育后cffDNA快速清除
│       └─ 验证：超声确认单胎活胎、随访cffDNA动态下降
│
└─ 临床因素
    └─ 近期流产/异位妊娠
        └─ 原因：前次妊娠男性胎儿cffDNA残留
        └─ 验证：询问病史、hCG动态监测、超声排除
```

#### 10.4.3 技术性矛盾的根因（板内/批次间差异）

| 现象 | 根因 | 判定方法 | 纠正措施 |
|------|------|---------|---------|
| 同一板内NTC仅1孔阳性 | 单孔污染（枪头/加样） | 查看该孔位置是否邻近强阳性 | 相邻孔重测，加强移液器消毒 |
| 多板连续NTC阳性 | 气溶胶污染或试剂污染 | 检查是否近期大量处理阳性样本 | 彻底清洁工作台、更换所有试剂 |
| 阴性对照（女性cfDNA）出现弱阳性 | 女性cfDNA含Y信号（微嵌合） | 扩大阴性对照库（>30例）确认 | 更换阴性对照来源（青春期前女性） |
| 阳性对照Ct漂移>2 | 标准品降解/探针失效 | 用新批次标准品平行测试 | 重配标准品、更换探针 |
| 整板内参Ct普遍偏高 | 循环条件出错（温度偏低） | 检查仪器温度校准 | 运行温度验证、仪器维护 |

### 10.5 矛盾结果报告规范

#### 10.5.1 报告分级制度

| 分级 | 代码 | 含义 | 临床处理 |
|------|------|------|---------|
| **确定性结果** | A | 3孔一致，无矛盾，内参正常 | 直接报告 |
| **高置信度结果** | B | 3孔一致但仅单靶标检出（TSPY+ SRY-@6周） | 报告+加注低丰度说明 |

| **无结果** | F | 3孔不一致且无法判别，或内参异常 | 发出"检测失败"通知，建议重新采样 |
